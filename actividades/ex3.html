What is a DTD file and what is their purpose?

Un archiu DTD es un document que defineix l'estructura d'un document XML.

How we can link a DTD file with a XML file?

!DOCTYPE element_arrel SYSTEM "archiu.dtd"

What is a XML-Schema file and why is better than DTD (put a little example)?

És un document que defineix l'estructura d'un document XML i es millor que DTD perqué DTD no suporta datatypes ni namespaces mentres que XML-Schema si.

How we can link a XML-Schema file with a XML file?

xsi:schemaLocation="localització de l'arxiu .xsd"

What is the DTD i XML-Schema files purpose?

Especificar elements i atributs utilitzats en un XML.
